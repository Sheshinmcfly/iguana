# Iguana

Landing page informativa sobre la iguana verde, un reptil fascinante de Latinoamérica.

## Tecnologías

- Bootstrap 5.3
- Bootstrap Icons 1.11
- Google Fonts — Raleway
- HTML5 semántico

## Secciones

- Hero con imagen a pantalla completa
- Descripción general del animal
- Características destacadas con iconos
- Galería de imágenes
- Footer con enlace externo
